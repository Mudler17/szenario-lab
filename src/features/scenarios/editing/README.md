# Editing-Bereich (Skeleton)

## Zweck
Dieser Bereich reserviert die künftige Struktur für eine getrennte Szenario-Bearbeitung innerhalb des Features `scenarios`.

## Aktueller Stand
- Aktuell existiert hier nur ein minimales Code-Skeleton.
- Es sind keine Formulare implementiert.
- Es gibt keine Speicherung.
- Es gibt keine Simulation.
- Es gibt keine Bewertung.

## Abgrenzung
Bestehende read-only-Komponenten bleiben unverändert und weiterhin ausschließlich für die Lesedarstellung zuständig.
