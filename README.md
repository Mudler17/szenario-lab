# szenario-lab

## Zweck
szenario-lab dient dazu, Organisationsszenarien strukturiert zu modellieren.

## Aktueller Status
Phase 4.4 · Szenario-Name, Szenario-Beschreibung und Szenario-Ziel im lokalen Draft bearbeitbar (mit Preview + Reset)

## Lokale Befehle
```bash
npm install
npm run dev
npm run build
```

## Hinweis
Die App erlaubt derzeit die Bearbeitung von drei Szenario-Grundfeldern im lokalen Draft: Szenario-Name, Szenario-Beschreibung und Szenario-Ziel.
Die Vorschau reagiert direkt auf alle drei Änderungen. Über Reset werden alle drei Originalwerte wiederhergestellt.
Wichtig: Keine Speicherung, kein LocalStorage, keine Simulation, keine OpenAI-Anbindung, keine weiteren aktiven Formularbereiche und keine neuen Module.
