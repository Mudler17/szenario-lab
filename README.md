# szenario-lab

## Zweck
szenario-lab dient dazu, Organisationsszenarien strukturiert zu modellieren.

## Aktueller Status
Phase 4.3 · Szenario-Name und Szenario-Beschreibung im lokalen Draft bearbeitbar (mit Preview + Reset)

## Lokale Befehle
```bash
npm install
npm run dev
npm run build
```

## Hinweis
Die App erlaubt derzeit die Bearbeitung von zwei Szenario-Grundfeldern im lokalen Draft: Szenario-Name und Szenario-Beschreibung.
Die Vorschau reagiert direkt auf beide Änderungen. Über Reset werden beide Originalwerte wiederhergestellt.
Wichtig: Keine Speicherung, kein LocalStorage, keine Simulation, keine OpenAI-Anbindung, keine weiteren aktiven Formularbereiche und keine neuen Module.
